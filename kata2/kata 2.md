## Kata 2 creacion de entorno virtual

Para empezar abrir el cmd y ubicar la carpeta donde yo quiero crear mi entorno virtual.

Una vez ubicada mi ruta escribir el comando:

- python -m venv env

para poder crear mis carpetas de entorno

Tras eso utilizar el siguiente comando importante de recordar el uso del backslash:

- env\Scripts\activate

Eso es lo que activara mi entorno, y todo se deberia de ver de la siguiente manera:
![[Captura.PNG]]

![[ ./resources/nombre-imagen.png]]

Finalmente solo vamos a ejecutar 2 comandos, uno para verificar las librerias instaladas y otro para poder instalar otra libreria, los cuales son:

- pip freeze (para poder ver las librerias)
- pip install (para poder instalar las librerias)
